React: Understanding hooks

Hooks allow us to create functional components with state, lifecycle, and other properties that were previously only possessable using classes. This allows us to write cleaner code and therefore more readable.

In this tutorial we will learn how to use two hooks: useState and useEffect.

useState

useState is the hook that allows us to add states to a function-created component.

To see how this hook works let's create a counter.

Primeiramente, vamos criar a estrutura básica do nosso componente:

Now import the useState hook:

import React, {useState} from "react";

And finally, let's create a state for our component:

const [counter, setConter] = useState(0);

Explanation:

Using destructuring, we assign the value zero to our state “counter” and a function “setCounter” that we will use to change the value of our state.

Once that's done, we just need to assign the function that changes the state in the "button" element and make the "p" element show the value of the state, that is:

<button onClick​={() => setConter(counter + 1)}>Contar </button>
<p>{counter} </p>

useEffect

If you are already familiar with using classes to create components, the useEffect hook can be compared to lifecycle methods (componentDidMount(), componentWillUnmount() and componentWillUpdate()).

useEffect takes two parameters: a function and an array of dependencies. Look:

useEffect(() => {}, [])

useEffect saves the passed function to execute it in one of three situations: whenever an element of the dependency array is updated, after the component is rendered, or after the component is disassembled.

Running whenever an element of the dependency array is updated

To execute the function whenever an element of the dependency array is updated, just add that element to the dependency array.

useEffect(() => {}, [element])

As an example, let's take the useState example. Let's update our page title according to the state counter value.

First we need to import useEffect:

import React, { useState, useEffect } from "react";

Now let's create the function that will change the page title and add the state counter to the dependency array:

useEffect(() => {
  document.title = `${counter} cliks`;
}, [counter]);

Running after component is rendered

To execute the function after the component is rendered (only once), just leave the dependency array empty.

useEffect(() => {}, [])

For example, let's create a component called Text and let's render it after three seconds. After it is mounted on canvas, the title of our page will be changed.

First let's create the Text component passing two parameters to useEffect. The first a function that will change the page title and the second an empty array.

function Text() {
  useEffect(() => {
    document.title = "O componente foi montado";
  }, []);
  return <p>O contador está visível!</p>;
}

Now let's recreate our Counter component by adding a “visible” state to it (with a default value of false) and a setTimeout function that will change the state value after 3 seconds:

const [visible, setVisible] = useState(false);
setTimeout(() => {
  setVisible(true);
}, 3000);

Finally, let's render the Text component only if the “visible” state has the value true:

return <div>{visible ? <Text /> : ""}</div>;

Running after component is disassembled

To execute a function after the component is disassembled, just return a function:

useEffect(() => {
  return () => {}
}, [])

To illustrate, let's do the opposite of the previous example. This time we are going to change the title of our page when the component is unmounted.

For this we will keep the structure of the previous example, making only three changes to our code.

First, let's change the value of state “visible” from false to true:

const [visible, setVisible] = useState(true);

Second, in the setTimeout function, let's change the value passed in setVisible from true to false:

setTimeout(() => {
  setVisible(false);
}, 3000);

Third, let's return, in useEffect, a function that changes the title of our page:

useEffect(() => {
  document.title = "O componente foi montado";
  return () => {
    document.title = "O componente foi desmontado";
  };
})
