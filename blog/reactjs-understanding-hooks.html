ReactJS: understanding hooks

Hooks allow us to create added components with state, life cycle and other properties that previously would only be possible using classes. This allows us to write cleaner and therefore more readable code.

In this tutorial, we will learn how to use the three main hooks: useState, useEffect and useContext.

useEffect

If you are already familiar with using classes to create components, the useEffect hook can be compared to the lifecycle methods (componentDidMount (), componentWillUnmount () and componentWillUpdate ()), although they work differently.

UseEffect takes two parameters: a function and an array of dependencies. Look:

UseEffect stores the passed function to execute it in one of three situations: whenever an element of the dependency array is updated, after the component is rendered, or after the component is unmounted.
