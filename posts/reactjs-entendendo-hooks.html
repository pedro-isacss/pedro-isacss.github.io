ReactJS: entendendo hooks

Os hooks nos permitem criar componentes funcionais com estado, ciclo de vida e outras propriedades que antes só seriam possessíveis utilizando classes. Isso nos permite escrever um código mais limpo e consequentemente mais legível.

Neste tutorial, aprenderemos a usar os três hooks principais: useState, useEffect e o useContext.

useState

O useState é o hook que nos permite adicionar estados a um componente criado com função.

Para ver como esse hook funciona vamos criar um contador.

Primeiramente, vamos criar a estrutura básica do nosso componente:

import React from "react";

function Counter() {
   return (
      <div>
        <button>Contar</button>
        <p>0</p>
      </div>
   );
}
 

export default Counter;

Agora importe o hook useState:

import React, {useState} from "react";

E finalmente, vamos criar um estado para o nosso componente:

const [counter, setConter] = useState(0);

Explicação:

Utilizando a desestruturação, atribuímos o valor zero ao nosso estado “counter” e uma função “setCounter” que utilizaremos para alterar o valor do nosso estado.

Feito isso, só precisamos atribuir a função que altere o estado no elemento “button” e fazer com que o elemento “p” mostre o valor do estado, ou seja: 

<button onClick​={() => setConter(counter + 1)}>Contar</button>
<p>{counter}</p>

Ao final, teremos o seguinte código:

import React, { useState } from "react";
	function Counter() {
	   const [counter, setConter] = useState(0);
	   
	   return (
	        <div>
                <button
	            onClick​={() => setConter(counter + 1)}>
	                Contar
	            </button>
	            <p>{counter}</p>
            </div>
	   );
	}
	 
export default Counter;

OBS: assim como no estado de classes, sempre que o estado for alterado, o componente será renderizado novamente.

useEffect

Se você já está familiarizado com o uso de classes para criar componentes, o hook useEffect pode ser comparado aos métodos de ciclo de vida (componentDidMount(), componentWillUnmount() e componentWillUpdate()), apesar de funcionarem de maneiras diferente.

O useEffect recebe dois parâmetros: uma função e um array de dependências. Veja: 

useEffect(() => {}, [])

O useEffect guarda a função passada para executá-la em uma de três situações: sempre que um elemento do array de dependências for atualizado, após o componente ser renderizado ou depois que o componente for desmontado.

Executando sempre que um elemento do array de dependências for atualizado

Para executar a função sempre que um elemento do array de dependências for atualizado basta adicionar esse elemento no array de dependências. 

useEffect(() => {}, [elemento])

Para exemplificar, vamos reaproveitar o exemplo do useState. Vamos atualizar o título da nossa página de acordo com o valor do state counter.

Primeiramente precisamos importar o useEffect:

import React, { useState, useEffect } from "react";

Agora vamos criar a função que mudará o título da página e adicionar o state counter ao array de dependências: 

useEffect(() => {
    document.title = `${counter} cliks`;
}, [counter]);

Ao final disso o nosso código será:

import React, { useState, useEffect } from "react";
function Counter() {
    const [counter, setConter] = useState(0);
	
    useEffect(() => {
        document.title = `${counter} cliks`;
    }, [counter]);
	
    return (
	    <div>
                <button onClick​={() => setConter(counter + 1)}>Contar</button>
	    </div>
    );
}
export default Counter;

Executando após o componente ser renderizado

Para executar a função depois que o componente for renderizado (apenas uma vez), basta deixar o array de dependências vazio. 

useEffect(() => {}, [])

Para exemplificar, vamos criar um componente chamado Text e vamos renderiza-lo após três segundos. Após ele ser montado em tela, o título da nossa página será alterado.

Primeiramente vamos criar o componente Text passando dois parâmetros para o useEffect. O primeiro uma função que irá alterar o título da página e o segundo um array vazio. 

function Text() {
    useEffect(() => {
        document.title = "O componente foi montado";
    }, []);
     
    return <p>O contador está visível!</p>;
}

Agora vamos recriar o nosso componente Counter adicionando a ele um estado “visible” (com um valor default false) e uma função setTimeout que alterará o valor do estado após 3 segundos: 

const [visible, setVisible] = useState(false);

setTimeout(() => {
   setVisible(true);
}, 3000);

Por último, vamos renderizar o componente Text apenas se o estado “visible” tiver o valor true: 

return <div>{visible ? <Text /> : ""}</div>;

O código final será:

import React, { useEffect, useState } from "react";
	

function Counter() {
    const [visible, setVisible] = useState(false);
    setTimeout(() => {
	    setVisible(true);
    }, 3000);
	return <div>{visible ? <Text /> : ""}</div>;
}
	

function Text() {
	useEffect(() => {
	    document.title = "O componente foi montado";
        }, []);
    return <p>O contador está visível!</p>;
}
	
export default Counter;

Executando depois que o componente for desmontado

Para executar a função após o componente ser desmontado, basta retornar uma função: 

useEffect(() => {
    return () => {}
}, [])

Para exemplificar, vamos fazer o contrário do exemplo anterior. Dessa vez vamos alterar o título da nossa página quando o componente for desmontado.

Para isso vamos manter a estrutura do exemplo anterior, realizando apenas três alterações no nosso código.

Primeiro, vamos trocar o valor do state “visible” de false para true: 

const [visible, setVisible] = useState(true);

Segundo, na função setTimeout, vamos alterar o valor passado no setVisible de true para false: 

setTimeout(() => {
    setVisible(false);
}, 3000);

Terceiro, vamos retornar, no useEffect, uma função que altere o título da nossa página: 

useEffect(() => {
    document.title = "O componente foi montado";
    return () => {
        document.title = "O componente foi desmontado";
    };

Ao final, teremos o seguinte código:

import React, { useEffect, useState } from "react";
	
function Counter() {
    const [visible, setVisible] = useState(true);    
    setTimeout(() => {
        setVisible(false);
    }, 3000);
        
    return <div>{visible ? <Text /> : ""}</div>;
}
        
function Text() {
    useEffect(() => {
        document.title = "O componente foi montado";
        return () => {
            document.title = "O componente foi desmontado";
        };
    }, []);
        
    return <p>O contador está visível!</p>;
}
        
export default Counter;

useContext

Esse hook tem como objetivo permitir o compartilhamento de dados entre os componentes.

Para exemplificar o uso desse hook nós vamos criar dois componentes diferente. Em um nós vamos incrementar o counter e no outro vamos mostrar o valor do counter. Além disso iremos criar um arquivo chamado “Context.js”, o qual será o responsável por compartilhar o valor do counter:

Criando o arquivo Context.js

Após criar o arquivo Context.js, faça as seguintes importações:

import React, { useState } from "react";

Agora crie o context:

const ConunterContext = React.createContext();

Por último crie o componente que irá prover os dados e exporte o “ConunterContext”: 

export function Context(props) {
    const [counter, setCounter] = useState(0);
    return (
        <ConunterContext.Provider value={{ counter, setCounter }}>
            {props.children}
        </ConunterContext.Provider>
    );
}

export default ConunterContext;

Explicação: com o componente “Context” você irá permitir/prover os dados e com o “ConunterContext” você irá buscar os dados.

Permitindo o acesso aos dados

No arquivo App.js, importe o “Context” e coloque o componente “Counter” dentro do provider “Context”: 

import React from "react";
import Counter from "./Components/Counter";
import { Context } from "./Components/Context";

function App() {
    return (
        <div className="App">
            <Context>
                <Counter />
            </Context>
        </div>
    );
}

export default App;

Criando o arquivo Text.js

Importe o useContext e “ConunterContext”:

import React, { useContext } from "react";
import ConunterContext from "./Context";

Crie o componente Text e use a desestruturação para buscar os dados necessários, no caso o counter: 

function Text() {
    const { counter } = useContext(ConunterContext);
    return (
        <div>
            <p>{counter}</p>
        </div>
    );
}

export default Text;

Criando o arquivo Counter.js

Faça as seguintes importações:

import React, { useContext } from "react";
import ConunterContext from "./Context";
import Text from "./Text";

Crie o componente Counter e dê acesso aos dados necessários, no caso o counter e o setCounter: 

function Counter() {
    const { setCounter, counter } = useContext(ConunterContext);
    return (
        <div>
            <button onClick​={() => setCounter(counter + 1)}>Contar</button>
            <Text />
        </div>
    );
}

export default Counter;
