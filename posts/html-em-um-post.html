HTML em um post

O objetivo é apresentar o html de forma rápida e direta.

Os elementos html

Os elementos html são formados por uma tag, pelos atributos e pelo conteúdo. Sintaxe: 

 < tag atributo="valor"> Conteúdo < /tag>
  • < tag > : indica onde o elemento começa (tag de abertura).
  • < /tag >: indica onde o elemento termina (tag de fechamento).
  • atributo: especifica uma determinada propriedade para o elemento.
  • Conteúdo: o que sera mostrado.

OBS: alguns elementos são formados sem tag de fechamento. Sintaxe:

Comentários

< !-- Aqui fica um comentário -- >

Criando um documento html

Crie um arquivo e adicione a extensão html. Exemplo: mypage.html

Estrutura básica de um documento html

< !DOCTYPE html >
< html >
    < head >
        < meta charset="utf-8" >
        < title >< /title >
    < /head > 
    < body >
    < /body >
< /html >
  • < !DOCTYPE html >: define o tipo do documento, no caso html.
  • < html >< /html >: ​indica o fim e o começo do documento html.​
  • < head >< /head >: aqui ficam as informações do documento html, como o tipo de caracteres (< meta charset=”utf-8" >) e o título da página (< title >< /title >).
  • < body >< /body >: dentro dessa tag ficarão todos os elementos que serão vistos em tela como textos, imagens, vídeos e outros.

Tags de contêiner

Um elemento contêiner serve para abrigar outros elementos.

  • < header >< /header >: abriga o cabeçalho da página.
  • < nav >< /nav >: abriga a navegação da página.
  • < div >< /div >: abriga um conteúdo da página.
  • < form >< /form >: abriga um formulário da página.
  • < footer >< /footer >: abriga o rodapé da página.

Tags de texto

  • < h1 >< /h1 > … < h6 >< /h6 >: definem títulos, subtítulos e tópicos.
  • < p >< /p >: define um parágrafo.
  • < span >< /span >: define um texto sem valor semântico.
  • < a href=”link” >< /a >: define um link.

OBS: uma tag < a > não necessariamente tem como conteúdo um texto. Também pode ser utilizada para colocar links em imagens, por exemplo.

Criando tabelas

Para construir uma tabela, utilizamos as seguintes tags:

  • < table >< /table >: define uma tabela.
  • < tr >< /tr >: define uma linha (table row).
  • < th >< /th >: define o título/cabeçalho de uma coluna (table header).
  • < td >< /td >: define os dados (table date).

Criando listas

Existem dois tipos de listas: as ordenadas e as não ordenadas.

Criando uma lista não ordenada:

  • < ul >< /ul >: define uma lista não ordenada.
  • < li >< /li >: define um elemento da lista.

Criando uma lista ordenada:

  • < ol type=”tipo de ordenação” >< /ol >: define uma lista ordenada.
  • < li >< /li >: define um elemento da lista.

Tags de Formatação

  • < br >: quebra de linha.
  • < b >< /b >: texto em negrito.
  • < i >< /i >: texto em itálico.
  • < sub >< /sub >: texto subscrito.
  • < sup >< /sup >: texto sobrescrito.
  • < mark >< /mark >: texto marcado.
  • < del >< /del >: palavra deletada.

Inserindo mídias

Inserindo uma imagem:

< img src="caminho" alt="texto alternativo" >
  • src: define onde a imagem está localizada. Pode ser o caminho até uma pasta ou o endereço da imagem na web.
  • alt: um texto que será exibido quando a sua imagem, por algum motivo, não puder ser carregada ou quando o usuário tiver alguma deficiência.

Inserindo um iframe:

< iframe src="endereço" >< /iframe >

OBS: um iframe incorpora uma outra página web na sua página.

Inserindo um vídeo:

< video controls >
    < source src="caminho" type="video/mp4" >
< /videos >
  • controls: adiciona os controles do vídeo (botão de play, volume…).
  • < source >: são as especificações do vídeo como o src (caminho para o vídeo) e o type (tipo do vídeo, geralmente mp4).
