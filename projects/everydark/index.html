EveryDark

EveryDark is an extension for Google Chrome that allows you to browse (on almost any website) with dark mode active.

GitHub Download

1 - Download and unzip the extension

Download the EveryDark extension to your computer by clicking here and then unzip it.

2 - Go to chrome://extensions/

Open a tab in your browser and go to the address chrome://extensions/.

3 - Enable Developer Mode

Enable developer mode so you can load new extensions.

4 - Load extension without compression

Finally, click on Load unpackage and select the EveryDark extension.
